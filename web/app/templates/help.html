Welcome to Online Ratings!

How does this work?

There are a lot of Go servers online that try to assign you a rating - KGS, OGS, IGS, Tygem, WBaduk, and countless others. The problem is that these ratings don't always agree with each other, or with the AGA's own rating system.

We're putting together a way for all these Go organizations to combine their game data, and calculate a single, more accurate rating.

For this to work, we need a way to put all of your accounts from OGS, KGS, etc. together.

Here's how it works:

  1. For each account you have, generate a new password
  2. Submit this password to the server (OGS, KGS, etc.)

That's it! Once the server knows who you are, they'll do the rest of the work in sending us the game data.

Head over to your account page to get started
